## 📌 Description

This is a simple **Dictionary Web Application** where users can search for a word and get its **meaning, phonetic pronunciation, synonyms, antonyms, and audio pronunciation**.
It fetches real-time data from an online dictionary API.

## 🚀 Features

* Search any English word
* Displays meaning and phonetic
* Shows synonyms and antonyms
* Audio pronunciation support 🔊
* Error handling for invalid words

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Dictionary API (https://api.dictionaryapi.dev/)

## ▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Run the file:
   dictionary.html

## 📷 Example

Input: hello
Output:
Word: hello
Phonetic: /həˈləʊ/
Meaning: A greeting used when meeting someone
Synonyms: hi, hey
Antonyms: goodbye
